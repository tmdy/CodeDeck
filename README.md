# CodeDeck

AI CLI 工具统一管理桌面应用 — Skills 管理 + Profile 配置启动器 + 模型映射 + 参数设置

## 技术栈

- **前端**：React 19 + TypeScript + Vite
- **桌面**：Electron 38
- **测试**：Vitest + Testing Library

## 开发

```bash
# 安装依赖
npm install

# 启动开发环境
npm run dev

# 类型检查
npm run typecheck

# 构建
npm run build

# 运行测试
npm test
```

## 打包

```bash
# Windows NSIS 安装包
npm run dist:win

# Windows 目录包（无需安装）
npm run dist:win:dir
```
